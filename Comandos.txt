git config --global core.editor "code --wait" (Seteo de editor de codigo predeterminado)
git config --global -e (ver archivo de configuracion local)
git config --global core.autocrlf true (para no tener problemas con los espacios al utilizar codigo externo)
git config -h (ver lista de comandos para configuraci√≥n)
clear (limpiar pantalla)
ls (para enlistar las carpetas que tengo en la carpeta actual)
ls -a (enlistar archivos incluyendo ocultos)
pwd (ver la carpeta en la que me encuentro)
cd + nombre de la carpeta a la que quiero ingresar (ingresar a carpeta especifica)
cd .. (para moverme a la carpeta anterior)
mkdir + nombre de carpeta (para crear carpeta)
mv + nombre de carpeta a mover + carpeta de destino (mover carpeta) 
git status (ver estado actual de nuestro repositorio)
git add + nombre de archivo que querramos subir (aceptar cambios en un archivo)
git commit -m "mensaje de cambio" (para hacer commit y comprometer nuestros cambios)
git commit (agregar comentario a cambios de archivos ya creados)
git rm + nombre de archivo (para eliminar archivo o carpeta. Se debe hacer commit para borrar en el master)
git restore --staged + nombre del archivo borrado (para resstablecer el archivo)
git restore + nombre del archivo borrado (se recupera el archivo)
git log --oneline (para ver el historial de cambios)
git pull (actualizar mi repositorio local de el que tengo en Github)
git branch NOMBRE-NUEVA-RAMA
git merge + nombre de rama (hacer merge de la rama al master)
git push (subir a la nube todos mis cambios)
git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY (clonar repositorio a directorio actual)
